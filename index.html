📖 使い方

現在地またはご自宅の住所を入力すると、釧路管内の最寄り津波避難場所を距離順に案内します。

  1. ヘッダーのシナリオ表示をタップして津波高さ・到達時間・潮位を設定（任意）
  2. 「現在地から最寄りの避難所を探す」または住所を入力して検索
  3. 地図上の丸マーカーをタップすると避難所の詳細が確認できます
  4. 検索結果をリセットするには GPS ボタン右の ✕ をタップ

このアプリは、釧路市・釧路町・白糠町・音別町の津波発生時の最寄り避難所を素早く確認するための防災支援ツールです。

1
まずこのボタンをタップ！

画面下の光るボタン 「📍 現在地から最寄りの避難所を探す」 をタップするだけで、GPS が現在地を取得し、最寄りの避難所を3ヶ所案内します。

2
地図上の避難所マーカーを確認する
  • 🔵 緊急避難場所（水色）：すぐ逃げる高台・建物
  • 🟢 避難所（緑）：しばらく生活できる施設
  • マーカーをタップ → 施設名・住所・標高を表示
  • 空=空き　混=混雑　満=満室
3
住所・地図タップでも検索できる
  • 🔍 住所入力：市町村＋番地を入力して「検索」
  • 🗺 地図タップ：ボタン後、地図の場所をタップ
  • 結果は近い順に3ヶ所。🔄 やり直すでリセット
4
シナリオを設定する（任意）

ヘッダーの 🌊 シナリオ表示 をタップで開きます。

  • 🌊 津波高さ：危険な避難所を自動除外
  • ⏱ 到達時間：到達圏を地図に表示
  • 🌊 潮位：干潮〜満潮で補正
5
メニューからページを切り替える

左上の ☰ をタップするとメニューが開きます。

  • 📖 使い方　📂 避難所データ
  • 🔒 プライバシーポリシー　👤 製作者情報
6
病院・通行止め情報を確認する
  • 🏥 病院マーカーをタップ → 標高・安全階数の目安
  • 🔴 赤丸 → 通行止め箇所。タップで詳細表示
⚠️ ご注意

本アプリはシミュレーションです。実際の避難行動は行政・防災無線の指示に必ず従ってください。

⚠️
浸水シミュレーションについて

釧路沖地震（震度7想定）を前提とした予想であり、国土地理院の標高データをもとにしたシミュレーションです。実際の浸水範囲・深さとは異なります。

最新情報は各市町村公式サイトまたは釧路市役所（0154-23-5151）でご確認ください。

📍
位置情報の取り扱い

取得した GPS 座標は端末内のみで処理します。外部サーバーへの保存・送信は一切行いません。アプリを閉じると位置情報は破棄されます。

🔍
住所検索の通信

住所検索時は入力テキストを国土地理院・OpenStreetMap へ送信します（個人情報は含みません）。

🗾
ルート表示の通信

ルート表示時のみ、出発地・目的地の座標を OSRM（経路探索サービス）へ送信します。

このアプリについて

釧路管内の住民が地震・津波の際に素早く避難所を確認できるよう、生成AIを活用して開発した防災支援ツールです。釧路市・釧路町・白糠町の避難所データを収録し、GPS や住所入力でご自身の最寄り避難所を瞬時に案内します。

管理者ログイン

避難所の空き状況・必要物資・メモを管理し、地図にリアルタイム反映します。

🔐 管理者モード
📖 管理者マニュアル
🔑 ログインと権限
情報更新モード（kushiro-tsunami）
  • 空き状況・必要物資・メモの入力と更新
  • 変更は地図にリアルタイム反映されます
全権限モード（kushiro-master）
  • 上記に加え、避難所の新規追加・編集・削除が可能
📋 空き状況・物資・メモを更新する
  1. リスト内の避難所名をタップして展開
  2. 空き状況ボタン（空き／混雑／満室）を選択
  3. 必要物資・メモを入力（任意）
  4. 「💾 保存」をタップ → 地図マーカーに即時反映

※ 空き状況・物資・メモをすべて空欄で保存すると設定がリセットされます。

➕ 避難所を新規追加する（全権限のみ）
🗾 マップから追加（推奨）
  1. 「マップ」タブに切り替え
  2. 「＋ 地図をクリックして追加」ボタンをタップ
  3. 地図上の追加したい場所をタップ → 緯度・経度が自動入力される
  4. 施設名・住所・種別などを入力して「💾 追加」
📋 リストから追加
  1. 「＋ 新規避難所を追加」ボタンをタップ
  2. 施設名・種別・市町村・緯度・経度を入力
  3. 緯度・経度はGoogle マップで場所を長押し → 数値をコピーして貼り付け
  4. 「💾 追加」をタップ
✎ 避難所情報を編集する（全権限のみ）
  1. リストで対象の避難所の ✎ ボタンをタップ
  2. 住所・種別・市町村・標高・収容人数などを変更
  3. 「💾 保存」をタップ
🗑 避難所を削除する（全権限のみ）
  1. リストで対象の避難所の 🗑 ボタンをタップ
  2. 確認ダイアログで「OK」→ 削除完了

※ 間違えて削除した場合は管理者にご連絡ください。

⚡ 全避難所のステータスを一括変更する（全権限のみ）
  1. 「リスト」タブの「全状況」フィルター右横のプルダウンから変更先を選択
  2. 「全適用」ボタンをタップ → 確認ダイアログで「OK」
  3. 全避難所のステータスが一括で更新され、地図に即時反映されます

※「不明（リセット）」を選ぶとすべてのステータスが未設定に戻ります。

🔴 通行止めマーカーを追加する（全権限のみ）
  1. 「マップ」タブに切り替え
  2. 「🔴 通行止めを追加」ボタンをタップ（カーソルが十字に変わる）
  3. 地図上の通行止めの場所をタップ → 説明入力フォームが表示される
  4. 説明を入力して「💾 追加」をタップ → 赤丸マーカーとして地図に保存

※ マーカーをタップして「🗑 削除」で削除できます。

🚰 給水所マーカーを追加する（全権限のみ）
  1. 「マップ」タブに切り替え
  2. 「🚰 給水所を追加」ボタンをタップ（カーソルが十字に変わる）
  3. 地図上の給水所の場所をタップ → 説明入力フォームが表示される
  4. 説明を入力して「💾 追加」をタップ → 🚰マーカーとして地図に保存

※ マーカーをタップして「🗑 削除」で削除できます。全ユーザーの地図にリアルタイム反映されます。
